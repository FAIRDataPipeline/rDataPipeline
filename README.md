The R implementation of the FAIR Data Pipeline API.

## Installation

In R:

```{r}
install.packages(devtools)
devtools::install_github("FAIRDataPipeline/rDataPipeline")
library(rDataPipeline)
```

Example usage can be found [here][SEIRS].
